PROJECT NAME:
MENTORFLOW X

BUILD MODE:
Build full project at once, BUT architect mentally + structurally in phases so no requirement, hidden pain point, feature, validation, or edge case is forgotten.

CORE IDEA:
We are building ONE final enterprise-grade product,
but phase structure exists as MEMORY + EXECUTION CONTROL.

========================================================
MASTER OBJECTIVE
========================================================

Build a premium, privacy-first, enterprise mentorship operating system using:

Frontend:
React.js (Vite) + Tailwind + Framer Motion + ShadCN

Backend:
Node.js + Express.js

Database:
MongoDB Atlas

Goal:
A stunning, scalable, evaluator-shocking platform that:
- Solves all PS requirements
- Solves hidden real-world mentorship pain points
- Prevents edge-case failures
- Has premium UI
- Has zero critical architecture mistakes
